# Stock Sell Net Profit Calculator (India)

A React + Vite web app that estimates realistic post-tax stock sell profit for Indian users.

## Features
- Delivery and intraday modes
- STCG/LTCG logic with holding period detection
- New regime income tax impact for intraday profit
- Detailed brokerage + statutory charge breakdown
- Hindi/English toggle
- Compare mode (delivery vs intraday)
- Downloadable PDF summary

## Run
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```
